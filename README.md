todolist -application
description:
cette application todolist permet de gerer une liste de taches de maniere simple et efficace.l'utilisateur peut ajouter,supprimer et marquer une tache comme termine en cliquant dessus.
le projet est realise sans framework uniquement html,css,javascript.
fonctionnalites:
ajouter une tache,suprimer une tache,marquer une tache comme effectue,supprimer toute les taches,interface responsive adapte au petits ecran.
